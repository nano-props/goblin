Features

A desktop control room for active Git work.

Goblin is built for juggling branches across multiple repos without losing terminal, editor, PR, or worktree context.

▣

Multi-repo tabs

Open local repositories or clone new ones, keep them in tabs, and restore the last workspace on launch.

⑂

Branch and worktree actions

Checkout, pull, push, create worktrees, open folders in editors, and reveal paths from one branch list.

⌁

PR and status signals

See sync state, dirty worktrees, PR review status, CI checks, and mergeability inline with each branch.

⌘

Keyboard-first navigation

Use j/k, arrows, repository shortcuts, and branch action keys. Customize the global shortcut or disable shortcuts entirely.

▤

Managed terminals

Run terminals inside the app, switch sessions, search output, replay terminal state, and keep context per repository.

◐

Themes and languages

Choose Goblin, shadcn/ui, or GitHub color themes, light/dark/system appearance, and English, Chinese, Korean, or Japanese.

Settings

One settings panel, every preference.

Dependency notes, keyboard shortcuts, app integrations, sync behavior, themes, language, and About now live in a single Goblin-styled settings window.

Workflow

Open. Inspect. Act.

Goblin keeps read-only context visible and puts high-frequency branch actions one keystroke away.

1

Open repositories

Open a local repo, clone a new one, or drop folders into the window. Goblin restores your workspace next time.

2

Inspect branch context

Review sync state, worktree status, commits, changes, PRs, checks, and terminal sessions without switching tools.

3

Act safely

Run common branch actions, open editors or terminals, copy patches, refresh resources, and keep destructive git work outside the app.
